    TV/ATV Remote Compatibility Matrix

    Search for the Full TV model Number in the Table below. Note the Type, Status Details and Remote Mode.

    Info

    If the screen type is not listed as hospitality there may be some caveats, such as not being able to clone configurations or use settings that appear in hospitality models. Your project manager will discuss this with you.

    Status - If listed as Incompatible, the TV will need to be replaced as there are fundamental issues in compatibility with our solution.

    Notes - Contains any details that need consideration.

    Remote Mode - CEC means the Apple TV can be hidden anywhere, IR means a line of sight between it and the remote control is needed.
