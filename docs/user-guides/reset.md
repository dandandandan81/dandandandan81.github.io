## Overview

Although we erase all guest data on checkout, some guests may be more security concious and waht to make sure their data is gone before they leave the room. 
This is exactly why we created the Reset app. Witha. few clicsk the Guest can begin the reset process and once complete, confiorm their logins and data have gone.

## Configuration
There are no user configurable componenets in the rest app.
